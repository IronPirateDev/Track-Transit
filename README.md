# TrackTransit 🚌

A modern bus tracking system for urban commuters.

## 🚀 Features

- Real-time bus tracking
- Route optimization
- Estimated arrival times
- User-friendly interface

## 🛠️ Installation
1.  ```bash
    git clone https://github.com/ironpiratedev/bus-tracking.git
    cd tracktransit
    npm install'
